La console Google Chrome

La console de Google Chrome est un outil développeurs qui nous permet d'exécuter et d'afficher du JavaScript à l'écran.
Il y a plusieurs façon d'y acceder :

  • Un clic droit sur l'écran et choisir "inspecter éléments"
  • Aller dans le menu "plus d'outils" et choisir "outils de développement"
  • Utiliser le raccourci clavier "ctrl + maj + i"

Une fois la fenetre ouverte choisir l'onglet console. Afin de visualiser sur notre console certaine valeur on va utiliser dans notre script JS l'instruction :

console.log(le nom de la (ou les) variables);

Où écrire le JavaScript

Comme pour le CSS on pourrai écrire notre JS sur notre page HTML dans un tag ‹script› ‹/script›, malheuresement cela deviendrai vite illisible, c'est pourquoi, comme pour le CSS nous allons l'écrire sur un autre fichier qui sera relié à notre page HTML.
Donc, dans le dossier ou est contenu notre page HTML nous allons créer un nouveau fichier .txt que nous allons renommer script.js .Nous allons relier ce fichier à notre page HTML en le mettant dans sont ‹head› comme ceci : ‹script src="script.js"›‹/script›
Ensuite on ouvre ce fichier dans le logiciel qui nous sert à créer notre code, c'est ici que nous écrirons notre JS.

La Syntaxe et les variables

Il est très important d'avoir un code lisible : ça permet de s'y retrouver, et donc d'éviter bon nombre d'erreurs. Pour cela comme pour la plupart des languages on utiliser l'indentation, elle consiste à "décaler" les instructions imbriquées. On utilise l’indentation pour hiérarchiser les instructions en plusieurs niveaux.

Les commentaires :

Il existe 2 manières d'écrire des commentaires en JS

  • Sur une seul ligne en commençant par //
  • Ou en encadrant sur plusieurs ligne son commentaire entre /* et */

On peut pour aller plus vite utiliser le raccourci ctrl/ . Encore une fois les commentaires nous servent à rendre notre code plus facilement lisible, pour nous, comme pour les éventuels collègues qui pourraient travailler sur le code avec nous.

Les variables :

Les variables servent à stocker des données. Pour créer (on dit déclarer) une variable, on utilise le mot-clé var, suivi du nom qu'on veut lui donner. Pour affecter une valeur à cette variable, on utilise le signe = . Il est possible d'affecter une valeur à une variable lors de sa déclaration.

Exemples :

Déclaration :
var age;
Affectation :
age = 18;
Tout en un :
var age = 18;

Une variable peut contenir différents types de données. Les nombres comme on vient de le voir ou des chaines de caractères. Une chaine de caractère, c'est... du texte. Les variables affecté de chaine de caractère doivent s'écrire entre guillmets.

Exemples :

var message = "Bonjour, ";

On peut grace au signe + , concaténer (mettre bout-à-bout) plusieurs chaine de caractères.

Exemples :

var message = "Bonjour, ";
var Homme = "Monsieur";
var Femme = "Madame";

console.log(message + Homme)
Cela affichera "Bonjour, Monsieur"

console.log(message + Femme)
Cela affichera "Bonjour, Madame"

On vient de voir que l'on pouvé ajouter des variables de type chaine de caractère, et on va voir maintenant qu'il est égalment possible d'interagir sur les variables nombres.
Il y a 5 opérations de bases :

  • l'addition(+)
  • la soustraction(-)
  • la multiplication(*)
  • la division(/)
  • le modulo (%)(c'est : le reste de la division euclidienne)

Exemples :

var x = 12;
var y = 3;
var result = x + y;

consol.log(result);

Cela affichera dans la console 15
De la même manière si l'on avait mis les autres opérateurs le resultat ce serait afficher.

Les Booléans

Nous venons de parler de deux types de variables : les nombres et les chaînes de caractères. Mais nous allons maintenant avoir besoin d'un troisième type de variables : les booléans. Ces variables ont deux valeurs possibles : vrai (true), ou faux (false).

On peut obtenir des booléens en comparant des valeurs. En effet, on dispose d'opérateurs de comparaison, qui sont au nombre de 6 :

  • L'opérateur d'égalité, == (attention, il y a bien deux signes "égal")
  • L'opérateur de différence, "est différent de", qui se note !=
  • L'opérateur "strictement plus grand que", ›
  • L'opérateur "strictement plus petit que",‹
  • L'opérateur "plus grand ou égal à", ›=
  • L'opérateur "plus petit ou égal à", ‹=.

Nous avons vu qu'on peut additionner ou multiplier des nombres et concaténer des chaînes de caractères.
Eh bien il existe aussi des opérations sur les booléans, au nombre de trois :

  • La négation, notée ! (si a est vrai, !a est faux, et inversement)
  • Le ET logique, &&. On a (a && b) qui est vrai si et seulement si a et b sont vrais
  • Le OU logique, || : (a || b) est vrai si et seulement si au moins l'une des deux valeurs est vraie.

Les conditions

Une condition est une sorte de « test » afin de vérifier qu'une variable contient bien une certaine valeur. Bien sûr les comparaisons ne se limitent pas aux variables seules, mais pour le moment nous allons nous contenter de ça, ce sera largement suffisant pour commencer. Voici la syntaxe d'une condition.

Exemples :

if(la condition à remplir (booléan))
{
l'ordre à executer
}

On peut voir dans l'exemple que notre condition et un booléan, c'est à dire que notre condition répond à la question (if) qui veut dire (si), donc si (if) ma condition est vrai alors execute l'ordre coder.
Il y a des compléments qui peuvent s'ajouter au si (if), c'est le (else) qui veut dire (ou), cela veut dire si (if) ma condition est vrai alors execute l'ordre coder ou (else) execute cette ordre. Cela s'écrit ainsi.

Exemples :

if(la condition à remplir (booléan))
{
l'ordre à executer
}
else
{
l'autre ordre à executer
}

Il existe un autre complément qui est le (else if) qui veut dire (ou si), c'est à dire admettons que ma première condition soit fausse, mais que ma seconde condition est vrai alors execute ce code.

Exemples :

if(la première condition)
{
l'ordre à executer
}
else if(la seconde condition)
{
l'autre ordre à executer
}

On peut rassembler les 3 marqueur de condition. Si la première condition et la seconde sont fausse alors dans ce cas execute ce code.

Exemples :

if(la première condition)
{
l'ordre à executer
}
else if(la seconde condition)
{
l'autre ordre à executer
}
else(comme les autres conditions sont fausse alors)
{
execute ce code
}

On peut inbriquer une condition dans une condition. Prenons l'exemple d'une limitatin de vitesse à 80km/h, pour qu'un message s'affiche sur votre tableau de bord, le programme va devoir s'interroger, afin d'afficher le bon message, si je roule à 80 ou un peu moins il me m'affichera "Tu roule à la bonne vitesse", si ma vitesse est inférieur à 50 par contre il va m'afficher "Acceleres un petit peu", ou si je roule à moins de 100km/h (donc entre 80 et 100) il affichera "Il faut que tu rallentisse un petit peu", et enfin si aucune de ces valeur n'est vrai (c'est donc que je roule forcement à plus de 100) alors il affichera "Tu roules beaucoup trop vite, c'est dangereux!".

Exemples :

var speed = 110 (l'indication du compteur)

if(speed‹80)
{
if(speed‹50)
{
console.log("Acceleres un petit peu");
}
else
{
console.log("Tu roule à la bonne vitesse");
}
}
else if(speed‹100)
{
console.log("Il faut que tu ralentisse un petit peu");
}
else
{
console.log("Tu roule beaucoup trop vite, c'est dangereux!");
}


Le switch

L'instruction switch commence par évaluer l'expression fournie (cette évaluation ne se produit qu'une fois). Si une correspondance est trouvée, le programme exécutera les instructions associées. Si plusieurs cas de figure correspondent, le premier sera sélectionné (même si les cas sont différents les uns des autres). Le programme recherche tout d'abord une clause case dont l'expression est évaluée avec la même valeur que l'expression d'entrée (au sens de l'égalité stricte). Si une telle clause est trouvée, les instructions associées sont exécutées. Si aucune case n'est trouvée, le programme recherche la case optionnelle default et si elle existe, les instructions correspondantes sont exécutées. Si cette case optionnelle n'est pas utilisée, le programme continue son exécution après l'instruction switch. L'instruction break peut optionnellement être utilisée pour chaque case et permet de s'assurer que seules les instructions associées à ce cas seront exécutées. Si on omet une instruction break, le script exécutera les instructions pour le cas correspondant et aussi celles pour les cas suivants jusqu'à la fin de l'instruction switch ou jusqu'à une instruction break.
Voici la syntaxe du switch.

Exemples :

switch(expression)
{
case1:
instructions1;
break;
case2:
instructions2;
break;
default:
intructions_def;
}

Les boucles (loop)

Les boucles permettent de répéter des actions simplement et rapidement. Il y a différents types de boucles mais elles se ressemblent toutes au sens où elles répètent une action un certain nombre de fois (ce nombre peut éventuellement être zéro). Les différents types de boucles permettent d'utiliser différentes façon de commencer et de terminer une boucle. Chaque type de boucle pourra être utilisé en fonction de la situation et du problème que l'on cherche à résoudre.

While (tant que) :

L'instruction while permet de créer une boucle qui s'exécute tant qu'une condition de test est vérifiée. La condition est évaluée avant d'exécuter l'instruction contenue dans la boucle. Une expression qui est évaluée avant chaque passage dans la boucle. Si cette expression est évaluée à vrai, instruction est exécutée. Si la condition n'est pas vérifiée, l'instruction n'est pas exécutée et le contrôle passe directement à l'instruction suivant la boucle.

Exemples :

while (condition)
{
instructions
}


Do...while (faire...tant que) :

L'instruction do...while permet de répéter un ensemble d'instructions jusqu'à ce qu'une condition donnée ne soit plus vérifiée. L'instruction est exécutée au moins une fois avant que la condition soit vérifiée. Quand la condition n'est plus vérifiée (vaut false ou une valeur équivalente), l'exécution de l'instruction do...while est stoppée et le contrôle passe à l'instruction suivante.

Exemples :

do
{
intructions
}
while (condition)


For (pour) :

L'instruction for crée une boucle composée de trois expressions optionnelles séparées par des points-virgules et encadrées entre des parenthèses qui sont suivies par une instruction à exécuter dans la boucle.

Exemples :

for(initialisation; condition; expression final)
{
intructions
}

Exemples :

function name(parameter1, parameter2, etc...)

code à executer;

Les arrays (les tableaux)

L'objet global Array est utilisé pour créer des tableaux. Les tableaux sont des objets qui servent de liste et possèdent plusieurs méthodes incorporées pour exécuter des opérations de parcours et de modification. Ni la taille d'un tableau ni le types de ses éléments n'est fixé. Puisque la dimension d'un tableau peut augmenter ou diminuer à tout moment. La numérotation des valeurs toujours à partir de zéro.

Créer un tableau

Exemples :

var name = [value0, value1, value2];

var fruits = ["Pomme", "Banane", "Orange"];

Les tableaux peuvent être stockés dans des variables et traités de la même manière que tout autre type de valeur, la différence étant que nous pouvons accéder à chaque valeur du tableau individuellement ou en groupe, il est possible d'afficher le nombre de valeur contenu dans la tableau avec length ou d'afficher le contenu du tableau lui même comme n'importe quels variable.

Afficher les valeurs du tableau

Exemples :

var fruits = ["Pomme", "Banane", "Orange", "Citron"];

for(var i = 0; i ‹ fruits.length; i++)
{
console.log(fruits[i]);
}

Cette exemple ce lit ainsi :

Voici mon tableau "fruits";
Pour une variable i qui sera égale à zéro; tant que i est inférieur à la longueur de mon tableau "fruits"; alors fait i + 1
Maintenant affiche la valeur i de mon tableau.
Et la chaque valeur de mon tableau s'affichera dans la console les unes en desous des autres.

Rajouter une ou des valeurs au tableau

Afin de rajouter une valeur à un tableau on va utiliser push.

Exemples :

fruits.push("kiwi");
Et kiwi ce rajoutera a la suite de notre tableau.


Enlever la dernière valeurs du tableau

Afin d'enlever la dernière valeur du tableau on va utiliser pop mais cette fois sans valeur.

Exemples :

fruits.pop();
Et kiwi sera enlevé de notre tableau.

Les objets

Un objet est un concept, une idée ou une chose. Un objet possède une structure qui lui permet de pouvoir fonctionner et d'interagir avec d'autres objets. Le JavaScript met à notre disposition des objets natifs, c'est-à-dire des objets directement utilisables. Vous avez déjà manipulé de tels objets sans le savoir : un nombre, une chaîne de caractères ou même un booléen.

Exemples :

var object = {
propertyName1: propertyValue1,
propertyName2: propertyValue2,
};
